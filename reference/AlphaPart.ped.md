# Sample pedigree for partition.

A dataset containing pedigree information and breeding values for six
individuals.

## Usage

``` r
AlphaPart.ped
```

## Format

A data frame with 6 rows and 8 variables:

- IId:

  individual's ID

- FId:

  Father's ID

- MId:

  Mother's ID

- gen:

  Generation

- country:

  Country

- gender:

  Individual's sex

- bv1:

  Breeding value for trait 1

- bv2:

  Breeding value for trait 1
